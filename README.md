# WEB API PERFUME - Frontend

## Descripción

Este proyecto es la aplicación frontend para interactuar con la API de Perfumes. La aplicación está construida con React utilizando Vite y está estilizada con Tailwind CSS. Permite a los usuarios probar los endpoints de la API de forma interactiva mediante un API Tester, además de ofrecer formularios para el registro y el inicio de sesión.

## Características

- **API Tester:** Interfaz para probar los diferentes endpoints de la API de Perfumes.
- **Formularios de Login y Registro:** Permite a los usuarios autenticarse y registrarse.
- **Notificaciones:** Feedback visual inmediato utilizando react-toastify.
- **Navegación:** Rutas configuradas con React Router.
- **Diseño:** Desarrollado con Tailwind CSS en tonos verdes.

## Tecnologías y Herramientas

- **React** con **Vite**
- **Tailwind CSS**
- **React Router**
- **Axios** para solicitudes HTTP
- **React Toastify** para notificaciones

## Estructura del Proyecto

- **src/components:** Componentes reutilizables (LoginForm, RegisterForm, EndpointTester, ResponseDisplay, etc.).
- **src/pages:** Páginas principales (Home, ApiTesterPage).
- **src/endpoints.js:** Definición de endpoints para el API Tester.
